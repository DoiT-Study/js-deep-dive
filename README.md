## 📌 규칙

### 정리 Format

- 책에 나온 목차 이름을 기준으로 잡고 정리한다.

### 정리 내용

- 내용 정리
- 읽고 느낀 점
- 다같이 이야기해보고 싶은 주제

### 정리본 업로드

- 폴더 구조 : ` {n}장-장-제목 / {자기이름}.md`
- 팀원의 정리본을 보고, 댓글로 의견을 남긴다
